About GiveReady

GiveReady is a free, open directory of more than 41,000 nonprofits across 29 cause areas. It is built to make charities discoverable and donatable by AI agents and humans alike. The directory is published as a website, a structured API, and an MCP server — so any donor, agent, or developer can search, filter, verify, and (where supported) donate to nonprofits worldwide.

Who runs it

GiveReady is operated by TestVentures.net, a fractional AI operations practice run by Geordie Wardman. Geordie is based in Switzerland and is the sole operator of the platform.

Contact: geordie@testventures.net

What we believe

Most nonprofits are invisible to AI agents. The agentic web is here, but the charitable sector lags every other domain in being legible to the LLMs that increasingly mediate where users send their money and attention. GiveReady exists to close that gap — to make 41,000+ verified organisations searchable, citable, and payable through whatever tool you happen to be using.

How it works

  • Search and discovery — by cause area, country, or keyword, via the website, the API, or the MCP server.
  • Structured profiles — every nonprofit has a profile with mission, programmes, impact metrics, and (where available) legal registration numbers.
  • Direct donations — through each nonprofit's own donation page (Every.org, the org's own site) or, where supported, USDC on Solana via the x402 protocol. GiveReady takes no fee and custody no funds.
  • Agent enrichment — AI agents can contribute missing data via a write-back API. Submissions are validated by consensus and credited on a public leaderboard.

Open source

The codebase is public at github.com/gswardman/giveready. The MCP server is published on npm as giveready-mcp and listed in the official MCP Registry as io.github.gswardman/giveready.
