# GiveReady Widget — Quick Start

Add a USDC donation button to your website in 30 seconds.

## The Code

Copy this and paste it into your HTML:

```html
<div id="giveready-donate" data-slug="finn-wardman-world-explorer-fund"></div>
<script src="https://giveready.org/widget/donate.js"></script>
```

Replace `finn-wardman-world-explorer-fund` with your nonprofit's slug.

That's it. The button will appear on your page.

## What It Does

1. Visitors see a "Donate with USDC" button
2. Click it → modal opens
3. Select amount ($1, $5, $10, $25, or custom)
4. QR code appears → scan with Phantom or Coinbase Wallet
5. Sign transaction in wallet
6. Donation settles on Solana (2-3 seconds)
7. 100% goes to your nonprofit (zero fees)

## Find Your Slug

Your slug is the URL-friendly version of your nonprofit name.

Examples:
- `finn-wardman-world-explorer-fund`
- `bridges-for-music`
- `the-wave-project`

Don't know your slug? Visit [giveready.org](https://giveready.org) or email [hello@giveready.org](mailto:hello@giveready.org).

## Requirements

- A Solana wallet address for receiving donations
- Your nonprofit registered in GiveReady database
- The ability to add HTML to your website

## Installation on Different Platforms

### Plain HTML
Just paste the code into any `.html` file.

### WordPress
1. Go to Pages → Edit
2. Switch to "HTML" editor
3. Paste the code
4. Save

### Webflow
1. Add "Embed" element
2. Paste the code
3. Publish

### Squarespace
1. Add "Code Block"
2. Paste the code
3. Save

### Wix
1. Add "Custom Code" element
2. Paste the code
3. Publish

### Other Sites
Look for "Custom HTML", "Code Block", or "Embed" option.

## Testing

1. Visit your website
2. Click "Donate with USDC"
3. Select $1
4. Scan QR code with your Phantom or Coinbase Wallet
5. Sign transaction
6. Done

Use Solscan to check your wallet: [solscan.io](https://solscan.io)
